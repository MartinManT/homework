Mājasdarba risinājums atrodams failā "ansible-playbook.yaml"

"hosts" faila saturs no mana datora /etc/ansible/hosts

"playbook-output.log" if tīrais outputs no playbook izpildes. 
Testēju ar 5x Debian 12.5 virtuālajām mašīnām lokālajā tīklā, norādot to IP adreses un hostname datora /etc/hosts failā.
